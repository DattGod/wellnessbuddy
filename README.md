WellnessBuddy — AI Personal Health & Wellness Coach
Overview

WellnessBuddy is a multi-agent AI personal coach that helps users plan meals, workouts, and sleep schedules. It leverages Google’s Gemini AI to provide personalized, actionable guidance for nutrition, fitness, and sleep, all in one platform.

Features

Personalized meal plans based on preferences and dietary restrictions

Custom fitness routines tailored to fitness level and goals

Sleep hygiene guidance and routines

Multi-agent AI system with context-aware memory

React-based chat interface for easy user interaction

Demo

Example queries:

“Suggest a 7-day vegetarian meal plan.”

“Create a 30-minute HIIT workout.”

“How can I improve my sleep quality?”

AI responses are tailored based on past interactions using MemoryBank.

Tech Stack

Frontend: React.js

Backend: Python, Flask

AI: Google Generative AI (Gemini-1.5)

Memory: In-memory session storage

Future Improvements

Streaming responses for real-time feedback

Long-term user progress tracking

Integration with fitness wearables

Weekly/monthly personalized analytics

Additional agents for mindfulness and mental wellness
